Hexo&Github博客搭建

Mar 26, 2020

如何正确食用Hexo&Github搭建属于自己的博客

你需要准备以下内容:

  • github没有则需要注册
  • node.js安装并配置环境
  • npm用来安装Hexo

第一步:在GitHub中创建一个仓库

在这里需要注意{yourusername}是你的github的用户名称

比如我的是GideonSenku.github.io 而你则是 {yourusername}.github.io

创建好以后,可以在你的Repository中看到这个仓库,点进去,点击Setting -> GithubPages -> Source

将Source的None改为Master

如果你看到以下内容则代表GitHub已经配置完成

第二步:下载安装Hexo

$ npm install hexo-cli -g
$ hexo init blog
$ cd blog
$ npm install
$ hexo server

打开127.0.0.1:4000,你会看到如下内容,说明Hexo已经安装完成

接下来将Hexo和Github关联起来,首先我们需要找到blog目录下的_config.yml这个文件,配置文件最下方

deploy:
  type: git
  repo:
    github: git@github.com:{yourusername}/{yourusername}.github.io.git
    #当然这里也可以用全地址(https://github/{yourusername}/{yourusername}.github.io.git)
  branch: master

我们还需要安装一个插件来将Hexo发布的文章push到Github

$ npm install hexo-deployer-git --save
$ hexo d -g
