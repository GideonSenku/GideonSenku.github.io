Mar 26, 2020

需求说明书

1. 引言

1.1 编写目的

编写该说明书是为了充分了解用户行为,以及系统的功能性和可行性分析

方便开发人员和管理人员作为参考资料。

1.2 背景

说明:

待开发的系统名称：一起练字官网（web端）

项目提出者：罗斌

项目开发者：罗斌(前端设计)、王天福(后台开发)

项目赞助者：上海米范文化传播有限公司

用户属性：一起练字、为你写诗粉丝,以及潜在用户,平台管理员

计算中心：阿里云

1.3 参考资料

HTML

CSS

JavaScript

ES6

Vue

NUXTJS

2. 任务概述

2.1 目标

开发一起练字官网的目的:

提升自媒体平台影响力、吸引更多兴趣爱好者、为用户提供新的信息获取渠道

开发一起练字官网的目标:

开发出可重构的一起练字官网,满足平台和用户的需要

用户能够访问平台信息,可直达淘宝店铺

尽量确保系统的安全性、稳定性

保证信息更新的及时性

一起练字官网的使用范围:

不限

2.2 用户特点

日常使用,一起练字、为你写诗关注爱好者 普通用户

对平台信息维护、更新,企业内部人员 管理员

可通过搜索引擎了解到该网站信息的用户 潜在用户

2.3 假定和约束

经费限制:

待定

开发期限:

争取2020年7月初完成优先级为1的设计

法律、法规和政策方面的限制:

包括但不限于《计算机软件保护条例》《网络安全法》等相关法律法规
